# Short Link Ninja

Сервис для создания коротких ссылок с личным кабинетом, учётом переходов и отчётами.

## Требования

- Docker и Docker Compose
- Make

## Быстрый старт

1. Клонируйте репозиторий и перейдите в каталог проекта.

2. Проверьте `.env.example` — файл уже содержит настройки для локального запуска в Docker. При необходимости скорректируйте значения и скопируйте его в `.env` (это также сделает команда ниже автоматически).

3. Поднимите окружение одной командой:

```bash
make test-up
```

Команда выполняет:

- остановку контейнеров и очистку volumes;
- создание `.env` из `.env.example`, если файла ещё нет;
- сборку и запуск контейнеров;
- `composer install`;
- генерацию `APP_KEY`;
- миграции базы данных.

4. Откройте приложение в браузере: [http://localhost:8080](http://localhost:8080)

## Первый вход

1. Перейдите на [http://localhost:8080/register](http://localhost:8080/register) и создайте пользователя.
2. После регистрации откроется раздел **Ссылки** — там можно создавать и удалять короткие ссылки.
3. Отчёт по переходам доступен в разделе **Отчёт**: [http://localhost:8080/report](http://localhost:8080/report)

Короткая ссылка имеет вид `http://localhost:8080/{short_code}` и перенаправляет на оригинальный URL. Каждый переход сохраняется в истории.

## Полезные команды

| Команда | Описание |
|---------|----------|
| `make test-up` | Полный подъём проекта с нуля |
| `make up` | Запуск контейнеров |
| `make down` | Остановка контейнеров |
| `make migrate` | Применить миграции |
| `make install` | `composer install` в контейнере |
| `make pint` | Форматирование кода (Laravel Pint) |

## Тесты

```bash
docker compose exec app php artisan test
```
